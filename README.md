### Map Reduce Implementation (Trab 2.3)
## Usage
### Get scripts
```
$ git clone https://github.com/dhanielo/nosql-map-reduce.git
$ cd nonsql-map-reduce
```
### Run Server
```$ python exercicio2.3.py```
### Run Client (in another terminal)
```$ python mincemeat -p nosql localhost```
